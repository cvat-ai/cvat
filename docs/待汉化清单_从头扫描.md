## 三、不可汉化（代码逻辑相关）

以下内容不应汉化，因为它们是代码逻辑的一部分：

1. **变量名/函数名/类名**：如 `trackerModel`, `Project`, `Model`
2. **枚举值/状态码**：如 `FINISHED`, `FAILED`, `STARTED`
3. **API 字段名**：如 `detail`, `message`, `status`
4. **CSS 类名**：如 `cvat-job-item`, `cvat-text-color`
5. **URL/路径**：如 `/requests`, `/analytics`
6. **技术术语（保留英文）**：如 `Webhook`, `API`, `OpenCV`
