# CVAT 中英对照表（总表）

本文件由 `docs/` 目录下的多份对照表/指南自动汇总生成，用于在正式汉化前统一查看“英文 → 中文”映射。

## 统计

- 汇总对照条目数（去重后）：877
- 其中：已标注文件：406，未标注文件：471
- 冲突（同一文件+英文对应多个中文）：0

## 使用方法（推荐）

- 优先按 `File` 找到目标文件，再在该小节内搜索英文原文。
- 对于 `File` 为空的条目：先在 `cvat-ui/src` 下用 `rg` 搜索英文原文定位真实文件。
- 必须保留占位符 `${...}` / `{...}` / `{{...}}` 与链接结构（只翻译可见文字）。
